### Kết quả đạt được tuần 1:

* Hiểu mô hình cloud computing AWS và nắm được các nhóm dịch vụ cốt lõi: Compute, Storage, Networking, Database, Security.
* Tạo và cấu hình tài khoản AWS Free Tier thành công, bật MFA cho tài khoản root.
* Quen với AWS Management Console — điều hướng dịch vụ, sử dụng thanh tìm kiếm, kiểm tra billing.
* Cài đặt và cấu hình AWS CLI gồm Access Key, Secret Key và Region mặc định.
* Thực hành AWS CLI: kiểm tra thông tin tài khoản, liệt kê region, mô tả EC2, tạo và quản lý key pair.
* Tạo thành công EC2 instance (t2.micro), kết nối SSH và gắn EBS volume.
* Hiểu IAM cơ bản: tạo IAM user, gán policy, cấu hình MFA cho tài khoản root.
